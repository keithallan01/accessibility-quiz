2. Which HTML element is best for marking up a main page heading?

3. A form field is missing a visible label. Which is the best accessible solution?

4. Why is it important to maintain visible focus indicators on links and buttons?

5. Which CSS technique is most appropriate for hiding content visually while keeping it available to screen readers?

6. When creating a custom button using <div>, which of the following is NOT required for accessibility?

7. Which ARIA attribute would you use to label an icon‑only button (e.g., a magnifying glass for “Search”)?

8. When creating a custom dropdown with JavaScript, what is the most important first step to make it accessible?

9. Which of these is the quickest way to test basic keyboard accessibility?

10. What is the main purpose of using semantic HTML elements (like <header>, <main>, <footer>)?
